Circle Packing
















A circle packing is an arrangement of circles in a given boundary such that no two circles overlap. Through the use of this algorithm a visualization of the defferent gene function levels according to their fold change numbers is made possible. There are two kinds of images possible a static image with different function labels and a dynamic image that allows the user to zoom into the circles to view functions in details.
